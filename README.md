# Projeto Prático: Dashboard de Disponibilidade Física (DF%) na Mineração 🚜📊
![Dashboard Final](image_ee723b.png)
Este repositório contém um projeto prático de Business Intelligence desenvolvido para o setor de Gestão de Ativos Industriais. O foco é transformar dados brutos de manutenção e operação em indicadores estratégicos para tomada de decisão.

---

## 🎯 Objetivo do Projeto
O objetivo deste dashboard é monitorar a saúde operacional de uma frota de equipamentos pesados. Ele permite que gestores de manutenção identifiquem rapidamente desvios de performance, analisem tendências e diagnostiquem as causas raízes de indisponibilidade, garantindo que a operação atinja suas metas de produtividade.

## 🛠️ Tecnologias e Ferramentas
* **Power BI:** Desenvolvimento de visualizações e interface de usuário (UX).
* **Power Query (Linguagem M):** Processo de ETL (Extração, Transformação e Carga), normalização de datas e cálculo de duração de eventos.
* **DAX (Data Analysis Expressions):** Criação de medidas calculadas para KPIs de disponibilidade, metas dinâmicas e variação temporal.

## 📊 Principais Funcionalidades
* **Análise de Tendência de DF%:** Monitoramento temporal (gráfico de área) para identificar oscilações históricas na performance.
* **Gestão vs Meta:** Comparação visual direta (velocímetro) contra a meta estabelecida de 90%.
* **Detalhamento de Indisponibilidade:** Gráfico de causa raiz focado nos principais sistemas (Motor, Hidráulico, Elétrico) que geram downtime.
* **Filtros Dinâmicos:** Segmentação granular por modelo de equipamento, categoria e período temporal.

## 🎨 Design (UX/UI)
O dashboard foi desenvolvido utilizando o conceito de **Dark Mode**, inspirado em salas de controle (CCO). O objetivo é reduzir a fadiga visual do usuário e destacar os KPIs principais através do contraste com cores de alerta (Laranja e Verde).
